Welcome to the demo site for sitelen lawa.

This is a Hugo theme for a personal website and/or blog based on the GOV.UK Design System. It makes use of GOV.UK Frontend, some components from HMRC Frontend, and the Inter UI and Font Awesome fonts.

This theme has the following features:

  • Multilingual
  • OpenGraph Metadata
  • Share Links (Clipboard, Facebook, Twitter, WhatsApp, LinkedIn, Tumblr, Web Share API)
  • XHTML Friends Network integration

In the future I'd like to add:

  • Structured data using Schema.org
  • Clean Insights integration
  • Content Warnings
  • gophermap generation
  • MARCXML generation
